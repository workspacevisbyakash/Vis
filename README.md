# ModernWeb - Multi-Page Website

A modern, responsive multi-page website built with HTML5, CSS3, and JavaScript. Features beautiful animations, interactive elements, and a professional design.

## 🚀 Features

- **Responsive Design**: Optimized for all devices (desktop, tablet, mobile)
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Interactive Elements**: Dynamic navigation, contact forms, FAQ sections
- **Performance Optimized**: Fast loading times with lazy loading and optimized assets
- **Cross-Browser Compatible**: Works on all modern browsers
- **Accessibility**: Keyboard navigation support and semantic HTML

## 📁 Project Structure

## 🚀 Getting Started

1. **Clone or download** the project files
2. **Open** `index.html` in your web browser
3. **Navigate** through the different pages using the navigation menu

### For Development

1. Set up a local web server (optional but recommended):
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx http-server
   ```

2. Open your browser and navigate to `http://localhost:8000`

## 🔧 Customization

### Colors
The main color scheme uses CSS custom properties. You can easily customize colors by modifying the gradient values in `main.css`:

```css
/* Primary gradient */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### Content
- Update text content directly in the HTML files
- Replace placeholder images with your own assets
- Modify contact information in `contact.html`
- Customize service offerings in `services.html`

### Styling
- All styles are contained in `styles/main.css`
- Responsive breakpoints: 768px (tablet), 480px (mobile)
- Modular CSS structure for easy maintenance

## 📧 Contact Form

The contact form includes:
- Client-side validation
- Required field checking
- Email format validation
- Success/error notifications
- Form submission simulation

**Note**: The form currently simulates submission. To make it functional, you'll need to:
1. Set up a backend server
2. Configure form processing
3. Update the form action and method
